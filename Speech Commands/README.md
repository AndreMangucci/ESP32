# ESP32 voice commands

## Requirements

Python Lib's

-pyaudio

-SpeechRecognition

## Basics

Voice commands sent from the script "voice_commands.py" via socket to the ESP32.
The "wifi_terminal.ino" does the simple interpretation of the string received.
